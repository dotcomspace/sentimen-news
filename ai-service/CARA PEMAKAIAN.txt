guys ini otak AI dan API-nya udah siap pakai. Total ada 4 file (economic_sentiment_model.keras, tokenizer.pkl, main_api.py, dan requirements.txt), pastiin ditaruh di satu folder yang sama ya.

Cara nyalain server AI-nya:

1. Buka terminal di folder itu, ketik: pip install -r requirements.txt

2.Di dalem file main_api.py, tolong isi bagian GEMINI_API_KEY pake API Key Gemini yang aktif.

3.Jalanin servernya pake command: uvicorn main_api:app --host 0.0.0.0 --port 8000

4.Kalau mau ngetes manual, buka browser ke: http://localhost:8000/docs

Daftar Endpoint buat ditembak dari Laravel (Guzzle/HTTP Client): Semuanya pakai method POST dan nerima Body JSON kayak gini: {"teks_berita": "teks berita ekonomi yang mau dianalisis..."}

1. Endpoint Prediksi Biasa: http://localhost:8000/predict Nanti response JSON-nya bakal balik kayak gini: {"status": "success", "teks_asli": "...", "sentimen": "Positif", "confidence_score": 92.5}

2. Endpoint + Fitur Rangkuman GenAI: http://localhost:8000/analyze-and-summarize Nanti response JSON-nya bakal balik kayak gini: {"status": "success", "sentimen": "Negatif", "rangkuman_ai": "Inti beritanya adalah...", "pesan": "..."}

Kabarin ya kalau ada error pas integrasi atau Cross-Origin (CORS)
